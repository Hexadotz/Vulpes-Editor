# Library/tools used
* [SFML 3.0.2](https://github.com/SFML/SFML): Used as the rendering pipeline.
* [ImGui](https://github.com/ocornut/imgui): for User Interface.
* [ImGui-SFML](https://github.com/SFML/imgui-sfml): used to link SFML and ImGui.
